# NewsApp


# Dictionary Web-App
### A simple news application to search for lastest news on trending issues around the world

## Installation
```
git clone https://github.com/WahabDev12/NewsApp.git

```
## Go to Project directory
```
cd NewsApp/flaskapp
```
## Install and activate virtual environment
```
pip install pipenv
pipenv shell

```
## Install requirements
```
pip install -r requirements.txt

```

## Start app on localhost
```
python app.py

```
